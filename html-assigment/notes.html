my 5 HTML tags

this is a list of 5 HTML tags i learned in the past weeks

p
For adding a paragraph of text we can use the p tag. The browser will add a space before and after the content of the paragraph.
sup
defines text as being superscript
h1
tag for the most important heading of the page
em
a string of emphasized, slightly more important text. Screen readers will change their voice for this text.
nav
the navigation of the page, this will contain the links to other pages of the website
